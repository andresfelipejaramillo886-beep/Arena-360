# SaaS Contadores (Flask + SQLite)

## Requisitos
- Python 3.10+

## Instalación
```bash
cd saas_contadores
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Ejecutar
```bash
./run.sh
```

o

```bash
python3 app.py
```

La app inicia en `http://localhost:5000`.
